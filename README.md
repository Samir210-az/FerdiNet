# FerdiNet - Fiber İnternet Provayder Saytı

Next.js + Firebase ilə qurulmuş tam funksional internet provayder saytı.

## Funksiyalar
- Müştəri şəxsi kabineti (balans, borc, ödəniş/bağlanma tarixləri)
- Admin panel (müştəri idarəetməsi, mesajlar, WhatsApp inteqrasiyası)
- Avtomatik email bildirişləri (bağlanmaya 3 gün qalmış)
- Sürət testi, əlaqə formu, tariflər

## Vercel-ə yükləmə
1. Bu reponu Vercel-ə import edin
2. Environment Variables əlavə edin (.env.local.example faylına baxın)
3. Deploy
